# GSE-ESP32

For Test NB-IoT (ESP32 Hardware Serial No 2)

PIN 16 (RXD2 ESP32) - BC95 (TXD) Green

PIN 17 (TXD2 ESP32) - BC95 (RXD) Blue

PIN 2 (IO2 ESP32)  -  BC95 (RES) Yellow
